About Inventory Slip Generator
Version {{ config.APP_VERSION }}

The Inventory Slip Generator is a powerful tool designed to streamline the process of creating inventory slips for cannabis products. It supports multiple data sources and formats, making it easy to generate professional inventory documentation.

Features
  • CSV Import: Import inventory data from CSV files
  • JSON Support: Process data from Bamboo and Cultivera JSON formats
  • API Integration: Direct integration with Bamboo and Cultivera APIs
  • Word Output: Generate professional Word documents
  • Theme Support: Multiple color themes for user preference
System Requirements
  • Modern web browser (Chrome, Firefox, Safari, Edge)
  • Internet connection for API features
  • JavaScript enabled
